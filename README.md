# OIBSIP-Task2-Unemployment_Analysis_With_Python

# Unemployment Analysis in India using Python

## 📌 Project Overview
This project focuses on analyzing unemployment trends in India using Python. The goal is to understand how unemployment rates have changed over time, especially during the COVID-19 period, and to explore regional and demographic differences.

---

## 📂 Datasets Used
1. **Unemployment_Rate_upto_11_2020.csv**
   - Includes state-wise data along with zone classification and geographical coordinates.

2. **Unemployment in India.csv**
   - Includes unemployment data with Urban and Rural classification.

---

## ⚙️ Technologies Used
- Python 
- Pandas
- Matplotlib
- Seaborn

---

## 🧹 Data Preprocessing
- Cleaned column names and removed unwanted spaces
- Converted date column into proper datetime format
- Handled missing values
- Created new features like:
  - Year
  - Month

---

## 📊 Visualizations Performed

### 1. Monthly Unemployment Trend
- Shows how unemployment changes over time on a monthly basis
- Helps identify spikes during COVID-19

### 2. Top States with High Unemployment
- Highlights states with the highest average unemployment rates

### 3. Zone-wise Comparison
- Compares unemployment across regions (North, South, East, West)

### 4. Correlation Heatmap
- Shows relationships between unemployment rate, employment, and labour participation

### 5. Impact of COVID-19
- Year-wise analysis to observe how unemployment increased during 2020

### 6. Geographical Distribution
- Visual representation using latitude and longitude

### 7. Urban vs Rural Comparison
- Analysis based on area classification from a second dataset

---

## 📌 Key Insights
- Unemployment increased significantly during 2020 (COVID period)
- Certain states consistently show higher unemployment
- Regional differences are clearly visible across zones
- Urban and rural unemployment patterns differ

---

## 🎯 Conclusion
This project shows how unemployment has changed over time and highlights the impact of COVID-19 on employment trends in India. It also demonstrates the importance of data cleaning, feature engineering, and visualization in data analysis.
